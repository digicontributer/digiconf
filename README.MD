DigiConf
============

DigiByte Core config generator. View online at https://powerful-cove-45486.herokuapp.com/

## Install
```bash
git clone https://github.com/digicontributer/digiconf && cd digiconf
npm install
npm start
```

## Prerequisites
- Node.js
- Python
